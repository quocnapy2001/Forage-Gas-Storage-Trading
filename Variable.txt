================================================================================
GAS STORAGE PRICING MODEL — VARIABLE REFERENCE
================================================================================

data_0.py
--------------------------------------------------------------------------------
natgas_df (DataFrame, -) : Raw data of natural gas dates and prices
prices (ndarray, £/MMBtu) : Observed monthly prices as a NumPy array
dates (Series, date) : Month-end observation dates
month_avg (Series, £/MMBtu) : Mean price by calendar month for seasonality plot

interpolation_1.py
--------------------------------------------------------------------------------
date_int (ndarray, days) : Observed dates as integer day counts for np.interp
price_vals (ndarray, £/MMBtu) : Observed prices aligned to date_int
query_date (str/date, date) : Input date for interpolate_price()

extrapolation_2.py
--------------------------------------------------------------------------------
start_date (Timestamp, date) : Earliest observation date; t=0 anchor for trend features
X (ndarray, mixed) : Design matrix [t, t^2, sin(2pi*m/12), cos(2pi*m/12)]
y (ndarray, £/MMBtu) : Target prices for Ridge regression
model (Pipeline, -) : Fitted sklearn Pipeline; StandardScaler + Ridge alpha=1.0
last_date (Timestamp, date) : Most recent observed date; boundary between interpolation and extrapolation
forecast_limit (Timestamp, date) : last_date + 730 days; maximum date accepted by estimate_gas_price()
discount_rate (float, annual %) : Continuous discount rate; default 0.0
valuation_date (Timestamp, date) : PV anchor date for discounting; defaults to today

pricing_prototype_4.py
--------------------------------------------------------------------------------
inj_dates (list[date], date) : Dates on which gas is injected
wdr_dates (list[date], date) : Dates on which gas is withdrawn
inj_vol (float, MMBtu) : Volume injected per injection event
wdr_vol (float, MMBtu) : Volume withdrawn per withdrawal event
inj_rate (float, MMBtu) : Maximum injection volume allowed per event
wdr_rate (float, MMBtu) : Maximum withdrawal volume allowed per event
max_vol (float, MMBtu) : Maximum physical storage capacity
storage_cost (float, £/month) : Fixed holding cost per calendar month between events
inj_cost (float, £/MMBtu) : Variable cost added on top of buy price at injection
wdr_cost (float, £/MMBtu) : Variable cost deducted from sell price at withdrawal
transport_cost (float, £/event) : Flat cost per injection or withdrawal event
storage_level (float, MMBtu) : Running inventory level; updated after each event
contract_val (float, £) : Running NPV of the contract across all events
months_passed (int, months) : Calendar months since last event; used to accrue storage cost
buy_price (float, £/MMBtu) : Estimated gas price at an injection date
sell_price (float, £/MMBtu) : Estimated gas price at a withdrawal date
event_log (list[dict], -) : One row per event; returned as the audit DataFrame
